Вывод: На панель HUB75 64x64 как базовый вариант.

Железо: Работать все будет на ESP32-S3 конкретно на плате Matrix Portal S3 Fly thinking
На чем будет программироваться: VSCode + PlatformIO + IDF
Нюансы: Основная цель обеспечить повторяемость последовательных замесов. Вес как таковой будет полезен, но не сильно обязателен, для него надо тарировать.
 - Тарировочный коэффициент предусмотреть по каждому датчику и общий.

```
ch1 = (raw1 - offset1) * scale1
ch2 = (raw2 - offset2) * scale2
ch3 = (raw3 - offset3) * scale3

total = ch1 + ch2 + ch3
weight = total * global_scale
```
